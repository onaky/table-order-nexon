# User Personas

## Persona 1: 김민수 (일반 고객)

| 항목 | 내용 |
|------|------|
| **이름** | 김민수 |
| **역할** | 매장 방문 고객 (일반) |
| **나이대** | 20~40대 |
| **목표** | 빠르고 편리하게 음식을 주문하고, 주문 상태를 확인하고 싶다 |
| **불편사항** | 직원을 부르고 기다리는 시간, 메뉴판이 잘 안 보이는 것, 주문이 제대로 들어갔는지 불안함 |
| **기술 수준** | 스마트폰/태블릿 사용에 익숙, 앱 사용 경험 풍부 |
| **사용 시나리오** | 테이블에 앉아 태블릿으로 메뉴를 탐색하고, 장바구니에 담아 주문한 뒤 상태를 확인 |

## Persona 2: 이지은 (건강 관리 고객 / 비건)

| 항목 | 내용 |
|------|------|
| **이름** | 이지은 |
| **역할** | 매장 방문 고객 (건강/식이 관리) |
| **나이대** | 20~30대 |
| **목표** | 메뉴의 재료와 칼로리를 확인하여 자신의 식단에 맞는 음식을 선택하고 싶다 |
| **불편사항** | 메뉴에 재료 정보가 없어서 직원에게 매번 물어봐야 하는 것, 알레르기 유발 재료 확인 어려움 |
| **기술 수준** | 디지털 기기 사용에 능숙, 건강 관련 앱 다수 사용 |
| **사용 시나리오** | 메뉴 상세 페이지에서 재료별 칼로리와 맛 프로필을 확인하고, 3D 뷰로 음식 구성을 파악한 뒤 주문 |

## Persona 3: 최영호 (진상 손님 / 까다로운 고객)

| 항목 | 내용 |
|------|------|
| **이름** | 최영호 |
| **역할** | 매장 방문 고객 (까다로운 유형) |
| **나이대** | 40~50대 |
| **목표** | 내가 원하는 대로 정확하게, 빠르게, 실수 없이 서비스 받고 싶다 |
| **불편사항** | 느린 응답 속도, 정보 부족으로 인한 재질문, 주문 누락/실수, 직원 호출해도 안 오는 것, 복잡한 UI, 내 주문이 잊혀진 느낌 |
| **기술 수준** | 스마트폰 기본 사용 가능하나 인내심 낮음, 한 번에 안 되면 짜증 |
| **사용 시나리오** | 메뉴를 보자마자 빠르게 결정하고 싶고, 주문 후에는 내 주문이 제대로 처리되고 있는지 계속 확인하고 싶음. 조금이라도 늦으면 불만. 정보가 부족하면 "이게 뭔데?"라며 화남 |
| **만족 조건** | ① 3초 이내 화면 로딩 ② 메뉴 정보가 충분해서 질문할 필요 없음 ③ 주문 상태가 실시간으로 보여서 "내 주문 어디갔어?" 할 일 없음 ④ 조작이 단순해서 헤매지 않음 ⑤ 주문 후 방치된 느낌이 들지 않음 |

## Persona 4: 박준호 (매장 관리자)

| 항목 | 내용 |
|------|------|
| **이름** | 박준호 |
| **역할** | 매장 운영자 / 관리자 |
| **나이대** | 30~50대 |
| **목표** | 실시간으로 주문을 모니터링하고, 메뉴를 효율적으로 관리하며, 매출을 파악하고 싶다 |
| **불편사항** | 주문을 놓치는 것, 메뉴 변경이 번거로운 것, 매출 현황 파악이 어려운 것 |
| **기술 수준** | 기본적인 PC/태블릿 사용 가능, 복잡한 시스템보다 직관적인 UI 선호 |
| **사용 시나리오** | 대시보드에서 실시간 주문을 확인하고 상태를 변경, 경과 시간이 긴 주문을 우선 처리, 메뉴 추가/수정/순서 변경 |
