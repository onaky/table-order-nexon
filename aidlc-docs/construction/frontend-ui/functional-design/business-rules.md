# Business Rules - Frontend UI

## 1. 장바구니 규칙

| 규칙 ID | 규칙 | 조건 | 동작 |
|---------|------|------|------|
| CART-01 | 최소 수량 | quantity < 1 | 항목 자동 제거 |
| CART-02 | 동일 메뉴 추가 | 이미 장바구니에 존재 | quantity + 1 |
| CART-03 | 총액 계산 | 항목 변경 시 | sum(price × quantity) 재계산 |
| CART-04 | 빈 장바구니 주문 방지 | items.length === 0 | 주문 버튼 비활성화 |
| CART-05 | 영속성 | 페이지 새로고침 | localStorage에서 복원 |
| CART-06 | 주문 후 초기화 | 주문 성공 시 | 장바구니 비우기 + localStorage 제거 |

---

## 2. 주문 상태 전이 규칙

### 상태 머신
```
pending → preparing → completed
  (대기중)   (준비중)     (완료)
```

| 규칙 ID | 규칙 | 설명 |
|---------|------|------|
| ORD-01 | 단방향 전이 | pending → preparing → completed (역방향 불가) |
| ORD-02 | 초기 상태 | 새 주문은 항상 'pending'으로 시작 |
| ORD-03 | 상태 표시 색상 | pending: 노란색, preparing: 파란색, completed: 초록색 |
| ORD-04 | 완료 주문 표시 | completed 상태는 체크 아이콘 + 흐린 스타일 |
| ORD-05 | 상태 변경 권한 | 관리자만 상태 변경 가능 (고객은 조회만) |

## 3. 인증 규칙

| 규칙 ID | 규칙 | 조건 | 동작 |
|---------|------|------|------|
| AUTH-01 | 토큰 만료 | API 401 응답 | 자동 로그아웃 + 로그인 페이지 이동 |
| AUTH-02 | 관리자 세션 | 16시간 경과 | 자동 로그아웃 |
| AUTH-03 | 테이블 자동 로그인 | localStorage에 유효 토큰 | 로그인 스킵, 메뉴 화면 직행 |
| AUTH-04 | 라우트 보호 | /admin/* 접근 시 토큰 없음 | AdminLoginPage로 리다이렉트 |
| AUTH-05 | 라우트 보호 | /customer/* 접근 시 토큰 없음 | CustomerLoginPage로 리다이렉트 |
| AUTH-06 | 토큰 첨부 | 모든 API 요청 | Authorization: Bearer {token} 헤더 자동 첨부 |

## 4. 경과 시간 색상 규칙

| 규칙 ID | 경과 시간 | 색상 | 스타일 | 의미 |
|---------|-----------|------|--------|------|
| TIME-01 | 0 ~ 5분 | 초록색 (emerald-400) | 기본 | 정상 |
| TIME-02 | 5 ~ 10분 | 주황색 (amber-400) | 기본 | 주의 |
| TIME-03 | 10분 이상 | 빨간색 (red-400) | 펄스 애니메이션 + 글로우 | 경고 |
| TIME-04 | 업데이트 주기 | 1초 | setInterval | 실시간 카운터 |
| TIME-05 | 표시 형식 | - | "방금 전", "3분 전", "1시간 전" | 상대 시간 |

## 5. 메뉴 관리 규칙 (관리자)

| 규칙 ID | 규칙 | 조건 | 동작 |
|---------|------|------|------|
| MENU-01 | 필수 필드 | 메뉴명, 가격, 카테고리 비어있음 | 저장 버튼 비활성화 + 에러 표시 |
| MENU-02 | 가격 범위 | price <= 0 | 유효성 에러 표시 |
| MENU-03 | 순서 변경 | 드래그 앤 드롭 완료 | 즉시 서버 저장 (낙관적 업데이트) |
| MENU-04 | 삭제 확인 | 삭제 버튼 클릭 | ConfirmModal 표시 후 확인 시 삭제 |
| MENU-05 | 이미지 업로드 | 파일 선택 | 즉시 업로드 → URL 반환 → 미리보기 표시 |

## 6. 테이블 관리 규칙 (관리자)

| 규칙 ID | 규칙 | 조건 | 동작 |
|---------|------|------|------|
| TBL-01 | 이용 완료 확인 | 이용 완료 버튼 클릭 | ConfirmModal ("정말 이용 완료 처리하시겠습니까?") |
| TBL-02 | 이용 완료 효과 | 확인 시 | 주문 → 이력 이동, 총액 0 리셋, 세션 종료 |
| TBL-03 | 주문 삭제 확인 | 삭제 버튼 클릭 | ConfirmModal ("이 주문을 삭제하시겠습니까?") |
| TBL-04 | 주문 삭제 효과 | 확인 시 | 주문 제거, 테이블 총액 재계산 |
| TBL-05 | 세션 시작 | 첫 주문 생성 시 | 자동으로 새 세션 생성 (서버 처리) |
